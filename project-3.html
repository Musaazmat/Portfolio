Project Overview

The Myntra Clone is an e-commerce platform developed using React.js and Node.js, featuring React Redux for state management. This project replicates the core functionalities of the popular fashion retail website Myntra. Key features include a dynamic product catalog and an interactive shopping cart where users can easily add and manage items. The application is designed with a responsive layout to ensure a seamless experience across all devices. The backend, built with Node.js, supports efficient data handling and secure while the frontend leverages React Redux to manage complex state and enhance user interactions.
